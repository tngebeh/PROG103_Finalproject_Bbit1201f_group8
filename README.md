# Freetown Community Health Centre - Queue Management System

This is our group final project for PROG103. It is a Python-based desktop application that helps manage patient queues efficiently at the clinic.

## Features
* Track patient arrivals and priority levels (Normal, Urgent, Emergency)
* Organize queues by department (General OPD, Maternity, Paediatrics)
* Automatic data persistence using a local JSON database
